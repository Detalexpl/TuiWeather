# TuiWeather
![LOGO](screenshots/logo.png)\
Simple TUI app showing weather.
# How to use 
## After lunching 
### press 's' to start typing names of cities 
![Zrzut ekranu_20260419_103200.png](screenshots/Zrzut%20ekranu_20260419_103200.png)
### next pres 'ENTER' (if after pressing enter you don't see name of your city in location widget on top use name of biggest city near you)
## if you don't like units TuiWeather is using
### press 'w' to open settings 
![Zrzut ekranu_20260419_103544.png](screenshots/Zrzut%20ekranu_20260419_103544.png)
### after choosing you units press 'ESC' and reload by pressing 'r'

# Installation
## best way for arch based Linux distributions
download from AUR

```https://aur.archlinux.org/tuiweather.git```

or if you are using yay
``yay -S tuiweather``

## best way
run:
``cargo install TuiWeather``
## second way 
download excutable from 
``
https://github.com/Detalexpl/TuiWeather/releases
``
